    Struct Database

    An database connection.

    This struct is a reference-counted wrapper around a sqlite3* pointer.

    Constructors

    Name Description
    this Opens a database connection.

    Properties

    Name Type Description
    changes [get] int Gets the number of database rows that were changed, inserted or deleted by the most recently executed SQL statement.
    errorCode [get] int Gets the SQLite error code of the last operation.
    handle [get] d2sqlite3.sqlite3.sqlite3* Gets the SQLite internal handle of the database connection.
    totalChanges [get] int Gets the number of database rows that were changed, inserted or deleted since the database was opened.
